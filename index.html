NPos3D: Nuclear Pixel Old School 3D

A Wireframe 3D Game Engine written in JavaScript and HTML5's 2D Canvas context.

The core goal of the NPos3D engine is to be easy to work with, requiring very little effort to start producing visual results, while providing many easily accessable configuration options for when their use is apropriate.

Documentation for NPos3D can be found Here.
